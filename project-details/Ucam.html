2016년에 명지대학교 컴퓨터공학과에서 주최한 모바일 앱 개발 경진대회에 출품한 앱으로, 캠퍼스에서 잉여롭게 우주 공강을 보내는 흑우들을 위한 앱이라는 컨셉으로 제작하였다. (당시 경진대회 공고) 사실 해당 GitHub 링크는 Git이 익숙하지 않았던 팀원들의 실수로 커밋이 꼬여버리자 다시 최종적으로 생성한 저장소로, 본래 프로젝트에 해당하는 저장소는 이 링크에 해당한다. Android 앱으로 제작되었으며, 당시에 새롭게 떠오르던 클라우드 데이터베이스인 Firebase를 같이 참여한 선배의 권유로 적용하게 되었다. (해당 저장소의 소유자이다.) 본 어플은 다음과 같은 기능들을 가진다:
  1. 간단한 로그인 기능

    Firebase를 이용한 로그인 세션 구현

  2. 같은 대학 캠퍼스의 학생들을 매칭

    회원정보에 기입된 대학 정보로 매칭

  3. 같은 공강 시간대 학생들을 매칭

    설정한 수업 시간표에 따른 공강 시간대 매칭

  4. 간단한 채팅 기능

    매칭된 학생들을 모아 채팅방 생성

지금 봐서는 정말 잘 모르겠지만, 쓸데없는 기능들이 많다고 까였다... 정말 억울했던 것은 시간이 매우 촉박해서 제대로 기능구현을 할 수 없었다는 점도 컸다. 시간이 얼마나 촉박했느냐면, 발표 전날에 선천적으로 앓고 있던 빈맥 수술을 위해서 병원에 입원했었는데, 그 병원에 비치된 태블릿으로 마지막 기능 구현을 위한 코드를 짰던 기억이 있다... 그래도 이 때가 대학교 1학년 겨울방학이었다는 점을 감안하면 나름 많이 배웠다고 생각한다. 그리고 무엇보다 수술을 마치고 나서 장려상을 받았다는 소리를 듣고 매우 기뻐서 웃었다가 출혈이 났던 기억도 있다.
