Node.js Express 서버로 구동하는 자동 진단서 작성 매크로이다. 가입된 회원의 이메일로 매일 새벽에 당일 명지대학교 캠퍼스 방문을 위한 진단서를 Puppeteer를 사용하여 자동으로 명지대학교 Myiweb에 들어가 작성한 진단서를 보내준다. AWS EC2 서버와 MySQL을 사용했다. 해당 앱의 웹사이트 UI에는 분명히 회원정보 수정을 위한 로그인 UI가 구현되어 있으나, 실제 로그인 기능은 구현되어 있지 않다. 과제 때문에 시간이 없었기 때문이다. 추후 수정 후 이 글을 수정할 예정이다. 이 앱의 기능은 다음과 같다:
  • 회원가입

    회원가입 웹UI에서 기입한 정보를 데이터베이스에 저장한다.

  • 진단서 작성

    Puppeteer를 사용하여 웹 크롤링으로 진단서를 작성한 후 진단 결과에 해당하는 출입증을 이미지로 저장한다.

  • 진단서 이메일 발송

    진단 결과에 해당하는 출입증을 회원의 이메일로 당일 새벽에 발송한다.

자세한 기능은 GitHub를 참고하기 바란다.
