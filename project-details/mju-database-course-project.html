전공 과목인 데이터베이스의 Term Project Final의 결과물이다. SQL 실습의 일환으로서 제작되었으며, Oracle DB와 JSP/Tomcat 서버로 제작되었다. 따로 서버를 띄우기 위한 프로젝트가 아니었기에 구동 환경은 내 랩탑 OS인 Windows 10에 해당한다. Term Project 조건으로서 최소 1개 이상의 입력을 받으며 Cartesian Product를 제외한 테이블 Join 구문을 활용한 SELECT문이 3개 이상 활용되었으며, 그 외에도 INSERT, UPDATE문 등도 활용되었다. 오해의 여지가 다분한 제목에서 보면 알 수 있듯이, 이 프로젝트는 애니메이션 스트리밍 사이트를 구축하기 위함이었다. 평소에 일본 애니메이션을 즐겨보는 나에게 여러 애니메이션 스트리밍 사이트의 검색 기능들을 보면서 아쉬운 부분들이 있었기에 해당 검색 및 조회 기능들을 보완하여 제작하였다. 기존 사이트들에도 잘 없는 기능을 추가한 예로는 평점별 그룹 조회 기능이 있겠다. 비디오들은 직접 DB에 저장되는 형태가 아닌 URL을 저장하는 형태로 하였으며, 각 비디오들은 Oracle DB SQL의 SYS_GUID() 함수를 통해 생성된 GUID로서 독립적으로 관리된다. 아쉽게도 회원가입과 로그인 기능은 넣지 못했으며, 대신 Oracle DB의 sqldeveloper를 통해서 계정 정보를 직접 DB에 삽입하였다. 이 사이트의 기능은 다음과 같다:
  1. 조회수순 애니 목록 조회

    사용자 조회수순으로 애니 목록을 조회할 수 있다.

  2. 애니메이션 비디오 스트리밍 페이지

    선택한 애니메이션을 시청할 수 있는 페이지로, 방문 시 해당 애니메이션에 대한 조회수가 증가한다.

  3. 애니메이션 업로드

    신규 애니메이션을 비디오 URL과 함께 업로드할 수 있다.

  4. 복합 검색 페이지

    제목, 장르별, 평점별, 완결 여부 등을 모두 고려하여 애니메이션을 검색할 수 있다.

자세한 사항은 GitHub에 있는 코드를 참고하기 바란다.
