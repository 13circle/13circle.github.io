Node.js Puppeteer를 사용해서 만든 웹 크롤러이다. 명지대학교 LMS 수강생들의 강의 동영상들을 강의별 및 강의 주차별로 디렉터리를 생성하여 다운로드해준다. 원래는 Node.js Electron 모듈로 GUI를 구현하려 했으나, 과제 때문에 매우 바빠서 하지 못했다. 따라서 해당 어플리케이션은 CLI 기반이다. 이름 그대로 강의 동영상을 다운로드하는 기능이 전부인 이 어플리케이션은 입력도 명지대학교 통합 로그인 학번과 비밀번호를 입력하는 것이 전부이다. 그럼에도 이렇게 회고를 하는 이유는 이 프로젝트를 제외하고도 Puppeteer를 즐겨 사용해왔음에도 명지대학교 LMS 사이트의 클래스 구조가 너무 중구난방이라서 크롤링이 힘들었던 기억이 있기 때문이었다. 한 가지 더 힘들었던 것은 (이것 때문에 GUI를 진지하게 고민했었다) CLI에서 비밀번호 입력란을 구현하는 것이었다. 이에 대한 코드를 짜면서 Node.js의 CLI 입력 기능이 얼마나 불편한지 새삼 다시 느끼게 되었다. 매우 별것이 없는 어플리케이션이기 때문에 자세한 설명은 GitHub 링크를 참고하기 바란다.
